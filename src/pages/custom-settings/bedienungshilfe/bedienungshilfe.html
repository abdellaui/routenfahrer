Bedienungshilfe Allgemein

Diese Anwendung vereinfacht es Ihnen eine Liste von Zielen abzuarbeiten. Erstellen Sie hier für eine Route. Daraufhin wird der Player aktiviert. Bei Start des Players werden Sie auf eine Navigation Ihrer Wahl weitergeleitet. Bei Ankunft kehren Sie zurück zu dieser Anwendung. Sie erhalten Informationen zu dem erreichten Ziel. Ebenso können Sie den Zustand des Zieles angeben. Im Anschluss können Sie mit der anstehenden Route fortführen.

Route

Ein Ziel besteht aus den folgenden Komponenten

Name: Name des Ziels wird primäre für die Visualisierung verwendet.

Erinnerung: Wird bei „Information zum Ziel“ angezeigt.

Adressinformationen: Die zufahrende Adresse.

Wiederholungen: Ermöglicht täglich wiederholende Ziele einmalig einzutragen.

Ziel hinzufügen

Hierfür können Sie auf der Seite „Route“ im obigen Bereich auf „Hinzufügen“ klicken. Ebenso können Sie eine Route importieren. Lesen Sie hierfür den Eintrag bei „Einstellungen / Route importieren“

Ziel de-/aktivieren

Ziehen Sie den Eintrag nach links. Klicken Sie auf „On/Off“. Beachten Sie dabei, dass dies ein Schalter ist. Ist der Eintrag eine aktive Route, so wird sie für die aktiven Tage deaktiviert. Ist die Route eine inaktive Route, wird sie für inaktive Tage aktiv. Ein manuell de-/aktivierter Eintrag erkennen Sie an der Markierung (blauer Kreis). Sie können unter „Einstellungen“ alle Markierungen entfernen und unter dem Punkt „Status von Zielen zurücksetzen“ die Einträge zurücksetzen.

Ziel löschen

Ziehen Sie den Eintrag nach links. Klicken Sie auf „Löschen“ und bestätigen Sie Ihren Vorgang.

Ziel bearbeiten

Ziehen Sie den Eintrag nach rechts. Klicken Sie auf „Bearbeiten“. Es öffnet sich der Editor. Speichern Sie Ihre Änderung.

Ziel starten

Ziehen Sie den Eintrag nach rechts. Klicken Sie auf „Starten“. Der Aktuelle Ziel wird aktualisiert.

Information zum Ziel

Halten Sie auf dem Eintrag gedrückt, bis im unteren Bereich sich ein schwarzes Fenster öffnet. Darin erhalten Sie die Informationen.

Karte Bedeutungen der Pin-Farben

Rot: Das sind Sie.

Blau: aktive Ziele.

Grün: erreichte Ziele.

Orange: inaktive Ziele.

 Einstellungen Nur anstehende Ziele anzeigen

Bei Wahl werden für heute inaktive Ziele in der Liste nicht angezeigt. Inaktive Ziele sind destotrotz bei der Umsortierung zu sehen.

Route importieren

Geben Sie die URL zur CSV Datei. Beachten Sie auf die Formatierung, diese erfahren Sie beim Exportieren. Vorkommende Fehler werden gemeldet. Sie können im Anschluss entscheiden, ob die Einträge angehängt werden oder ihre Liste überschreiben.

Route als CSV exportieren

Sie können ihre Route als CSV exportieren. Die erste Zeile enthält dabei die Attributsnamen. Attributsnamen mit einem „_“ Präfix sind Pflichtfelder.

Status von Zielen zurücksetzen

Alle manuell de-/aktivierten Ziele werden in ihren unveränderten Zustand zurückgesetzt.

Alle Ziele löschen

Beachten Sie hierbei, dass sie einen Backup machen, indem Sie Ihre Liste im Voraus exportieren, denn dieser Vorgang ist nicht rückkehrbar.

Automatisch fortführen

Bei Abarbeiten eines Zieles wird das nächste Ziel automatisch gestartet. Sie müssen also nicht zusätzlich auf das „Play“ Button klicken.

Automatisch zurücksetzen

Sie können eine bestimmte Uhrzeit eingeben. Bei Neustart ab der Uhrzeit werden erreichte Ziele zurückgesetzt. Dies betrifft nicht den Status eines Zieles.

Navigation

Es werden verfügbare Navigationsanwendungen aufgelistet. Mit Ihrer Wahl setzen Sie die Standartanwendung für die Navigation. Grundsätzlich ist keine Navigation notwendig, denn Sie können diese Anwendung als Checklist verwendet.
